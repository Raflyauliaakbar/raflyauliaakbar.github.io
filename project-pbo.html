Aplikasi Pendeteksi Cuaca

Proyek ini merupakan Aplikasi Pendeteksi Cuaca berbasis Java yang dibangun menggunakan konsep Pemrograman Berorientasi Objek (PBO). Aplikasi ini dirancang untuk memberikan informasi cuaca terkini seperti suhu, kelembapan, serta kondisi langit (cerah, berawan, hujan) dengan antarmuka sederhana. Program ini mengintegrasikan data cuaca (API atau data simulasi) ke dalam logika aplikasi, sehingga dapat membantu pengguna mengetahui kondisi cuaca secara cepat dan mudah.

Key Features

  • Antarmuka sederhana menggunakan Java Swing sehingga mudah digunakan.
