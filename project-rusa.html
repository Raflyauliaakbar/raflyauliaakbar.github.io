Deteksi Rusa

Project Deteksi Rusa adalah bagian dari repo Belajar-Computer-Vision yang bertujuan untuk mendeteksi keberadaan rusa dalam gambar menggunakan teknik Computer Vision dan Deep Learning. Proyek ini menggunakan dataset khusus yang dibuat dengan anotasi bounding box dan dilatih menggunakan algoritma deteksi objek modern seperti YOLO.

Key Features

  • Model YOLO digunakan untuk deteksi objek secara real-time.
  • Visualisasi hasil deteksi langsung pada gambar input.
  • Evaluasi performa menggunakan metrik mAP, precision, dan recall.
